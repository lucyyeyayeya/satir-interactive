## 這個工具是什麼？

一個即時互動工具，用於薩提爾工作坊中：

- 主持人出題（例如「聽完這個故事，你的第一個提問是什麼？」）
- 所有學員同時在自己的手機輸入回答
- 主持人公佈所有人的答案，一起討論
- 每輪結束後進入下一輪，課堂結束後可複製完整歷程摘要

**適合場景：** 20–25 人的小型工作坊，多輪提問循環，強調提問脈絡的呈現。

## 課前準備（主持人）

### ⏱ 建議課前 10 分鐘完成

### 第一步：暖機（必做）

工具的伺服器在閒置一段時間後會進入休眠，第一個連線需要等待約 10–20 秒。

1. 打開工具網址：https://f5370e35.satir-interactive.pages.dev
2. 等待頁面完全載入
3. 看到「教師儀表板」畫面即代表伺服器已喚醒 ✅

> 💡 **就像課前先開好投影片** — 讓伺服器先準備好，學員進來就不用等。

---

### 第二步：建立（或進入）課堂

**第一次使用 / 新課堂：**

1. 點擊右上角「**＋ 新建課堂**」
2. 輸入課堂名稱（例如：`5/26 薩提爾工作坊`）
3. 點擊「**建立課堂**」
4. 自動進入題庫編輯頁

**已建立過的課堂：**

1. 在儀表板找到課堂名稱
2. 點擊「**✏️ 題庫**」進入編輯，或「**▶ 課堂**」直接進入主持頁

> 💡 課堂建立後會記錄在你的瀏覽器中，下次打開網站還會在。但伺服器的題目與狀態若重啟會消失，建議每次課前重新確認題目是否存在。

---

### 第三步：設定題目

在「**題庫編輯**」頁面：

1. 點擊「**＋ 新增題目**」
2. 輸入每一輪的提問（例如：`聽完故事後，你的第一次提問是什麼？`）
3. 可用 ▲▼ 調整順序，× 刪除
4. 建議每個題目就是一輪提問的主題

> 💡 題目即是「提示語」，引導學員這一輪要提交什麼。

---

### 第四步：測試連線（選做，強烈建議）

1. 在題庫編輯頁，點右上角「**▶ 進入課堂**」
2. 用自己的手機掃描等待畫面中的 **QR Code**，確認可以成功加入
3. 確認主持頁可以看到學員人數出現

---

### 課前確認清單

- [ ] 已開啟工具網址，伺服器已喚醒
- [ ] 課堂名稱已建立
- [ ] 所有輪次的題目已輸入
- [ ] QR Code 可以正常掃描加入
- [ ] 電腦或平板螢幕可以讓所有人看到（若有投影）

## 課堂流程（主持人）

### 整體流程圖

```
題庫編輯 → 開始課堂 → [第 N 輪：等待回答 → 公佈答案] → 結束課堂 → 複製摘要
```

---

### 一、讓學員加入

1. 在題庫編輯頁點「**▶ 進入課堂**」
2. 課堂主持頁會出現 **QR Code** 和加入連結
3. 讓學員掃描 QR Code 或輸入連結加入
4. 右側「學員名單」會即時更新人數

> 💡 可以把 QR Code 投影在螢幕上，或點「**QR / 連結**」放大讓大家掃。

---

### 二、開始課堂

確認學員都加入後：

1. 點 Header 的「**✏️ 題庫**」返回題庫編輯頁
2. 點底部「**▶ 開始課堂**」按鈕
3. 自動跳轉到主持頁，顯示第一題

---

### 三、每一輪的循環

```
顯示題目 → 學員回答 → 等待 → 主持人公佈答案 → 討論 → 下一題
```

**作答階段（主持人畫面）：**
- 顯示目前題目文字
- 即時顯示「已回答 N / 總人數 人」
- 等大多數人回答後，點「**公佈答案**」

**公佈後（主持人畫面）：**
- 所有學員的回答卡片展開顯示
- 可開啟「**顯示暱稱**」切換匿名或具名顯示
- 和學員一起討論各個回答
- 討論結束後點「**下一題 →**」

**最後一題公佈後：**
- 按鈕顯示「**結束課堂**」
- 點擊進入摘要頁

---

### 四、主持頁操作提示

| 按鈕 | 功能 |
|------|------|
| **← 儀表板** | 返回課堂列表（課堂繼續進行，可重新進入） |
| **QR / 連結** | 開啟放大版 QR Code + 複製連結 |
| **✏️ 題庫** | 返回題庫編輯（可在等待階段新增題目） |
| **☰（漢堡）** | 收合 / 展開右側學員名單欄位 |
| **顯示暱稱** | 切換答案卡顯示「暱稱」或「參與者 N」 |

## 學員操作

### 加入課堂

1. 用手機掃描主持人螢幕上的 **QR Code**
   - 或直接打開主持人提供的連結（格式：`https://...pages.dev/join/XXXX`）
2. 輸入自己的**暱稱**（可自訂，重複時系統自動加 ② ③ 後綴）
3. 點「**進入課堂**」

### 提交回答

1. 等待主持人開始課堂
2. 看到題目後，在文字框輸入回答
3. 點「**送出**」（可多次送出，每次都會記錄）
4. 等待主持人公佈答案

### 注意事項

- 若不小心關掉瀏覽器，重新開啟連結、輸入暱稱即可重新加入
- 送出後可以繼續補充，再次點送出即可
- 公佈答案後無法再送出（等下一題）

## 課後摘要

課堂結束後會自動進入**摘要頁**，顯示每一輪的題目與所有學員的回答。

> ⚠️ **摘要不會自動儲存！** 頁面關掉就消失。

### 複製摘要步驟

1. 點「**📋 複製摘要**」按鈕（頁面頂部或警示橫幅）
2. 打開筆記 App（備忘錄、Notion、Word…）貼上
3. 確認內容完整後再關閉頁面

### 複製後的格式範例

```
【課堂摘要】房間 XXXX
════════════════════════

第1題：聽完故事後，你的第一次提問是什麼？
────────────────────────
• 小明：這件事發生時，你當下的感受是什麼？
• 小華：是什麼讓你選擇那樣回應？
• 阿志：你希望對方聽到什麼？

第2題：第二次提問，你想問什麼？
────────────────────────
• 小明：...
```

## 常見問題

**Q：學員掃 QR Code 後一直轉圈？**
→ 伺服器可能還在喚醒中，等 10–20 秒後重新整理。下次課前記得先暖機。

**Q：學員進來後主持人看不到人數增加？**
→ 確認主持頁的右上角綠點（已連線），若顯示橘色轉圈表示連線中，稍候即可。點 ☰ 展開學員名單確認。

**Q：題目存了卻消失了？**
→ 伺服器可能重啟過。重新進入題庫編輯頁，若局部還原失敗，重新輸入題目後點空白處即可儲存。

**Q：可以在課堂中途新增題目嗎？**
→ 可以。在 `answering`（作答中）階段，點 Header 的「✏️ 題庫」可返回編輯頁新增，但新題目會排在後面。

**Q：學員手機斷線怎麼辦？**
→ 重新打開連結、輸入相同暱稱重新加入即可，之前的回答仍保留在伺服器上。

**Q：可以同時有多個老師使用嗎？**
→ 可以，每個課堂有獨立的 4 碼房間代碼，互不影響。

**Q：課堂結束後可以再看嗎？**
→ 不行，目前課堂結束後若關閉頁面就無法再查閱。**課堂結束請立即複製摘要。**
